## Краткое описание
При вводе суммы перевода `0` сервис принимает перевод и показывает успешное уведомление.

## Окружение
- ОС: Windows 10
- Браузер: Chrome/Edge (актуальная версия)
- URL: `http://localhost:8000/`

## Severity / Priority
- **Severity:** Critical (нарушение бизнес-логики: перевод нулевой суммы не имеет смысла)
- **Priority:** High

## Предусловия
- Открыта главная страница приложения
- Выбран счёт «Рубли»
- Введён валидный номер карты (16 цифр)

## Шаги воспроизведения
1. Выбрать счёт «Рубли».
2. В поле «Номер карты» ввести `1111 1111 1111 1111`.
3. В поле «Сумма перевода» ввести `0`.
4. Нажать «Перевести» (если кнопка доступна).

## Фактический результат
Появляется успешное уведомление вида: «Перевод 0 ₽ на карту … принят банком!».

## Ожидаемый результат
Перевод должен быть отклонён:
- кнопка «Перевести» не должна отображаться/быть активной при сумме ≤ 0, **или**
- должно отображаться сообщение об ошибке, например «Сумма должна быть больше 0».
